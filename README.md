"# Fitness-Class-Booking-System-" 
Pleased to share a project I recently completed as part of my Computer Science studies at Islamia College University of Peshawar.

I designed and developed a Fitness Class Booking System — a desktop application built in Python that solves a real-world problem: replacing manual, error-prone paper-based booking systems with a reliable digital solution.

📌 Technical highlights:

• OOP Architecture — a User base class with Member and Staff subclasses demonstrating inheritance, encapsulation, and method overriding
• Tkinter GUI — a 4-tab interface for viewing classes, booking sessions, managing bookings, and admin controls
• JSON Persistence — all class and booking data is saved and reloaded between sessions
• Input Validation — guards against empty fields, invalid capacity, duplicate bookings, and over-capacity errors
• Test Suite — 10 structured test cases covering both standard operations and edge cases, all passing
• HTML Demo — a browser-based interactive version of the system built with vanilla HTML, CSS and JavaScript

Building this end-to-end — from class design to GUI to testing to deployment demo — gave me a much clearer picture of how real software projects are structured.
